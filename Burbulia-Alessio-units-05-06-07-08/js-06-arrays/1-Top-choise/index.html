1-Top-choise

Create an array to hold your top choices (colors, pets, books, whatever). For each choice, log to the screen a string like: "My #1 choice is blue."

Bonus: Change it to add the correct number suffix, e.g. "My 1st choice, "My 2nd choice", "My 3rd choice", "My 4th choice", etc.

watch on console. Change the scripts on index.html.
